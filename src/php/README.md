# 概要

# カフェ（喫茶えすぽわぁる）のオーダーサイト

<br>

## 基本機能

-   ユーザー登録機能
-   ログイン/ログアウト機能
-   商品一覧/ログアウト機能
-   カート機能
-   注文機能
    <br>
    <br>

## 追加機能

-   注文履歴機能
-   ページング機能
-   合計金額自動反映機能
-   クレジットカード WebAPI 連携機能
-   注文確定メール送信
-   パスワードハッシュ化機能
-   ログイン認証フィルター機能
    <br>
    <br>

## 独自実装

-   管理者機能
-   スタンプ収集機能
-   商品出品・編集伊能
-   リコメンド機能
-   プロフィール情報編集機能
-   CSV ダウンロード機能

    <br>
